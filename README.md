# student-performance-ml

# 🎓 Student Performance Prediction (ML Project)

## 📌 Overview
This project predicts student marks based on study hours and attendance using Machine Learning.

## 🚀 Features
- Predicts student performance
- Uses Linear Regression
- Includes data visualization ( scatter plot)

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Visualizations
- Scatter Plot: Study Hours vs Marks
- Scatter Plot: Attendance vs Marks
- Scatter Plot: Actual vs Predicted

## 🧠 ML Model
- Linear Regression

## 📈 Results
- Achieved good accuracy (R² Score ~0.8+)
- Found positive correlation between study hours and marks
